# <img src="icon.png" width="32" height="32" style="vertical-align: middle;" /> Dev Starter

A minimal reference runtime that boots Avocado OS with development tools, SSH access, and i2c-tools pre-installed. No application code or build scripts — just a blank canvas to start building on.

- Boot any supported target with a ready-to-use development environment
- SSH in as root with an empty password for quick iteration
- Pre-installed i2c-tools for hardware bring-up and peripheral debugging
- Use as a starting point for your own application or as a base to layer extensions onto
